# Generador de CV

## 🧩 Paso 1: Definir el proyecto

### Proyecto: Chatbot generador de CV

### ❓ ¿Qué es el proyecto?

El proyecto consiste en una **aplicación web desarrollada con Laravel** que funciona como un **chatbot**.

El chatbot guía al usuario mediante una serie de preguntas (nombre, perfil profesional, experiencia, estudios y habilidades) y, con esa información, **genera automáticamente un currículum vitae** que se muestra en pantalla.

El objetivo del proyecto es ofrecer una forma sencilla y guiada de crear un CV básico desde el navegador.

### ✅ ¿Cuál es el MVP (Producto Mínimo Viable)?

El MVP del proyecto es:

- Una aplicación web en Laravel
- Una interfaz simple tipo chat (pregunta → respuesta)
- Preguntas básicas:
    - Nombre completo
    - Perfil profesional
    - Experiencia laboral
    - Estudios
    - Habilidades
- Generación de un **CV en texto** mostrado en una vista web

👉 Si esto funciona, **el proyecto ya es válido y está completo como MVP**.

### ⭐ ¿Qué cosas son “buenas de tener”?

(No obligatorias para la primera versión)

- Exportar el CV a **PDF**
- Plantillas de CV (Junior / Técnico / Creativo)
- Guardar el CV en base de datos
- Posibilidad de editar respuestas
- Mejorar el diseño visual del chat
- Soporte para varios idiomas
- Uso de IA para mejorar la redacción del perfil

### 🏁 ¿Cuándo estará completo el proyecto?

El proyecto estará completo cuando:

- El chatbot haga todas las preguntas del MVP
- El usuario pueda responder desde la web
- El CV se genere correctamente y se muestre en pantalla
- La aplicación funcione sin errores críticos

> Todo lo que venga después es mejora, no obligación.
